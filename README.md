# Evaluation and Experiment Tracker Explainer

An interactive Streamlit explainer for AI engineering evaluation, experiment tracking, and production observability.

The content is adapted from the local deck `Copy of AI Engineering Foundations.pptx`. The app turns the deck into a hands-on teaching path with short explanations, mental models, interactive controls, and a small experiment tracker lab.

## Pages

1. Evaluation Metrics
2. What to Evaluate in AI Systems
3. Designing an Evaluation Pipeline
4. Experiment Tracking
5. Observability and Architecture
6. Monitoring vs. Observability
7. Experiment Tracker Lab

## What It Implements From the Deck

- Traditional metrics and their limits
- AI-specific evaluation dimensions
- Component-level evaluation pipeline design
- Evaluation rubrics and datasets
- Experiment run logging and comparison
- MLflow and Weights & Biases concepts
- AI engineering architecture layers
- Monitoring versus observability
- Latency, cost, drift, traces, logs, and user feedback

## Run Locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

The Streamlit Cloud main file path is:

```text
streamlit_app.py
```
